## Devlog

That line of code changes the color of the cat sprite. The period is used for separation. The word before the period is what object you're changing, the word after the period is what specific thing within that object you're changing. For example, in this line, you're changing the modulate inside of the \_sprite. 



All I can figure out is that the modulate changes the color of the sprite. I'm guessing that modulate is basically a modifier, because it has mod in it.
